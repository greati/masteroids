# Introdução #

O projeto consiste em um jogo de naves espaciais que precisam proteger
a galáxia. Tem como função a obtenção da nota da 3º unidade das disciplinas de EDB1 E LP1.

# Guia de utilização(MODO TEXTO) - Linux #
	
	1 - Descompactar o projeto na pasta de preferência
	1 - Abra o terminal
	2 - make
	3 - Para executar:
		bin/Masteroid
	4 - Para limpar os arquivos de compilação:
		make clean
